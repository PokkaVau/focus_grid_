# FocusGrid 🎯

> **Stay focused. Get things done. Your productivity, structured.**

FocusGrid is a complete, production-quality productivity web application. It features a modern, clean, minimal SaaS-style UI that helps users manage daily tasks, track habits, plan weekly goals, and monitor progress without the noise of overly complex project management tools.

## 🚀 Features

*   **Task Tracker:** Add, delete, and complete tasks with filtering options (All / Pending / Done).
*   **Habit Tracker:** Build routines with a daily habit checker and a streak tracking system (🔥).
*   **Weekly Planner:** Plan your week at a glance with a responsive 7-day grid layout.
*   **Progress Dashboard:** A real-time, animated progress bar visualizing your daily completion percentage, accompanied by motivational quotes.
*   **Daily Reset System:** Automatically detects when a new day begins, resetting daily habits while safely preserving your streaks.
*   **Dark Mode Toggle 🌙:** A smooth, premium theme transition between light and dark aesthetics.
*   **Fully Responsive:** Designed mobile-first using CSS Grid, ensuring a flawless experience across phones, tablets, and desktops.
*   **Privacy First:** Operates entirely client-side. No databases, no external servers. Your productivity data is stored securely in your browser's LocalStorage.

## 🛠️ Built With

*   **HTML5** (Semantic structuring)
*   **CSS3** (Variables, Grid, Flexbox, Animations)
*   **Vanilla JavaScript** (ES6+, strict mode, DOM manipulation, LocalStorage)
*   **FontAwesome** (Icons)
*   **Google Fonts** (Inter Typography)

## 🔐 Security & Architecture

Even without a backend, FocusGrid follows strict frontend security practices:
*   No `eval()` or unsafe JavaScript execution.
*   Strict DOM manipulation (`textContent` & `createElement` only) to prevent XSS (Cross-Site Scripting).
*   Safe JSON parsing for LocalStorage data persistence.
*   Keyboard shortcut blocking to prevent unauthorized local inspection.

## 💻 Running Locally

To run FocusGrid locally, simply open the `index.html` file in any modern web browser. 

No build steps, no package managers, and no local servers are required!

## 🌐 Deployment

FocusGrid is ready to be deployed instantly on Vercel or GitHub Pages. 

### Vercel Deployment
1. Upload this repository to your GitHub account.
2. Log in to [Vercel](https://vercel.com/) and click **Add New... > Project**.
3. Import your FocusGrid repository.
4. Click **Deploy** (No build commands or configuration needed).
